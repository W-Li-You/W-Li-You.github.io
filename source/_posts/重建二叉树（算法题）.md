**题目：**

        输入某二叉树的前序遍历和中序遍历的结果，请构建该二叉树并返回其根节点。

        假设输入的前序遍历和中序遍历的结果中都不含重复的数字。

<!--more-->

**示例：**

![](https://img-blog.csdnimg.cn/f943c4611c1e4803ac87b455a1b187ef.png)

        Input: preorder = \[3,9,20,15,7\], inorder = \[9,3,15,20,7\]  
        Output: \[3,9,20,null,null,15,7\]

**代码：**

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    int[] preorder;
    HashMap<Integer,Integer> in=new HashMap<>();
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        this.preorder=preorder;
        for(int i=0;i<inorder.length;i++){
            in.put(inorder[i],i);  //方便获取根节点在中序遍历的坐标
        }
        return hui(0,0,inorder.length-1);
    }
    TreeNode hui(int root,int left,int right){ //递归方法
        if(left>right) //递归结束的条件
            return null;
        TreeNode node=new TreeNode(preorder[root]);  //创建节点
        int index=in.get(preorder[root]); //获取根节点坐标
        node.left=hui(root+1,left,index-1); //获取左子树
        node.right=hui(root+index-left+1,index+1,right); 
        //root+index-left+1为当前节点+左子树个数+1
        return node;
    }
}
```

** 总结：**

        前序遍历时先是根节点，然后是左子树，右子树。

        中序遍历先是左子树，然后是根节点，最后是右子树。 ![](https://img-blog.csdnimg.cn/6b5088b985e8419b9dace85aa9e8b803.png)

        前序遍历第一个数是二叉树最上面的根节点，遍历inorder，找到根节点的位置。根节点左边是左子树的中序遍历结果，右边是右子树的遍历结果。先找左子树，在左子树的遍历结果里最上面的一个根节点就是前序遍历的第二个数，以此类推，采用递归的方式解决问题。
